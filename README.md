# AI-Powered Social Media Automation using Make.com

## Overview
This project automates social media content creation and publishing using Make.com and Generative AI. 
It generates platform-specific posts for LinkedIn, Facebook, and Telegram from a single content input.

## Problem Statement
Manually creating and posting content on multiple social media platforms is time-consuming and inconsistent.

## Solution
An AI-driven automation workflow that:
- Fetches content from Google Sheets
- Uses Generative AI to summarize and personalize content
- Automatically posts to multiple platforms using routing logic

## Tools & Technologies
- Make.com
- Google Gemini AI
- Google Sheets
- LinkedIn API
- Facebook Pages API
- Telegram Bot
- Prompt Engineering

## Workflow
1. Content link or summary is added to Google Sheets
2. Make.com triggers the workflow
3. Gemini AI summarizes the content
4. Router splits flow into three branches:
   - LinkedIn
   - Facebook
   - Telegram
5. Platform-specific AI prompts generate optimized posts
6. Posts are automatically published

## Key Learnings
- AI prompt engineering
- Automation workflows
- Routing and conditional logic
- Real-world AI use cases
